- X共有文生成
- レスポンシブ対応

## 公開前に確認すること

- 非公式表記が分かりやすい場所にある
- 公式画像・ロゴ・ライバー画像を掲載していない
- 公式プロフィール由来のデータであることを明記している
- データ更新手順が残っている
- スマホ表示で横はみ出しがない
- `npm run test` と `npm run build` が通る
